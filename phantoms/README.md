# Phantom Design & Fabricaton

Here are resources for developing modular implant phantoms like the one presented in the TONIQ paper.

For each design folder:
- All commercial products used are collected in product-links.md
- The printing folder contains all printing targets needed for fabrication
- The solidworks folder contains all source design files to facilitate adapting the design, e.g. for different metal components

Stanford users may also find useful an (incomplete) summary of [campus resources](stanford-resources.md) for phantom developement.
